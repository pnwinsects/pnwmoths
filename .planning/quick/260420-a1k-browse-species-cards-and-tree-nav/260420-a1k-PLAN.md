# Quick Task: Browse Species Cards and Tree Navigation

## Goal
Two UX improvements to the browse page:
1. Clicking a species image at family/subfamily level opens the tree to that species' genus
2. When a genus is expanded, show species as large image cards (one image per species, responsive: full-width mobile, 2-column desktop)

## Must-Haves
- [ ] `_expandToSpecies(speciesSlug)` method navigates tree to the species' genus
- [ ] Family-level and subfamily-level image strips are clickable (buttons wrapping images)
- [ ] `taxon.js` adds `navImage` property to each species (best image by navigational flag + weight)
- [ ] `_renderSpecies` shows a responsive grid of cards with images
- [ ] Mobile: 1 column (full width); ≥600px: 2 columns
- [ ] Images use aspect-ratio 376/249, object-fit: cover

## Files to Change
- `src/_data/taxon.js` — add navImage per species
- `src/components/pnwm-taxon-browser.js` — expand-to-species + species cards
