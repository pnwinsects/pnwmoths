# Milestone v2.0: Glossary Tooltips

**Status:** ✅ SHIPPED 2026-04-23
**Phases:** 19–21
**Total Plans:** 5

## Overview

Species prose automatically highlights the first occurrence of each glossary term at build time with a tooltip/popover showing the definition and optional CDN image. The feature works without JavaScript (native `<abbr title>` fallback) and does not pollute the Pagefind search index (definitions live in `data-*` attributes).

## Milestone Summary

**Key Decisions:**

- Use `node-html-parser` (not JSDOM/cheerio) for build-time text-node traversal — ~10x faster, zero native dependencies
- Load glossary.csv at Eleventy startup via `csv-parse` (existing dependency) — not DuckDB, to avoid second DuckDB lifecycle
- Initialize `seen` Set inside the transform callback per invocation, never at module scope
- Use `escapeRegex` on all terms before constructing RegExp — terms like `1A+2A`, `W-mark`, `CuA1` break without escaping
- Sort glossary terms longest-first before matching — prevents "wing" from partially matching inside "forewing"
- `substituteTerms()` uses while-loop with pos cursor — wraps all unseen terms in one text-node pass (gap fixed in 19-04)
- Native HTML Popover API (`popover="auto"`) — no external tooltip library; Escape + click-outside dismiss for free
- Definition text in `data-definition` attribute — never in DOM at build time, so Pagefind ignores it
- Per-term popover elements injected at runtime; `getBoundingClientRect()` positioning below the `<abbr>`
- Phase 21 folded into Phase 20 — all TIP-02 (CDN images in popover) delivered in Phase 20 rewrite

**Issues Resolved:**

- substituteTerms() dropped positionally-earlier shorter terms when a longer term appeared later in the same text node — fixed with while-loop in 19-04
- Popover flash on show (popover positioned before showPopover() call) — fixed via off-screen pre-position
- hideTimer shared across term instances causing incorrect dismiss behavior — fixed by scoping per-term

**Issues Deferred:**

- CSS Anchor Positioning (TIP-04) — Baseline 2026, defer to future milestone when cross-browser
- GLOS-07 (plural/morphological variant matching) — requires stemming or synonym entries in glossary.csv
- Fix close button on lightbox — unrelated todo, carry forward
