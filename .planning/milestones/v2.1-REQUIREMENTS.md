# Requirements Archive: v2.1 Species Fact Sheet Gaps

**Archived:** 2026-05-20
**Milestone goal:** Close the remaining UX and feature gaps between pnwmoths and the reference pnwinsects-app on species fact sheet pages.

---

## v2.1 Requirements (All Complete)

### Chart Improvements
- [x] **CHART-01**: User sees axis labels on the phenology chart — X-axis labeled "Month", Y-axis labeled "# Records" — ✅ Phase 22
- [x] **CHART-02**: Phenology chart Y-axis begins at 0 and scales to the highest monthly record count — ✅ Phase 22

### Photo Navigation
- [x] **PHOTO-01**: User sees a thumbnail strip below the main species image; clicking any thumbnail selects it as the main displayed image — ✅ Phase 23
- [x] **PHOTO-02**: Thumbnail strip replaces dot navigation for multi-image species (dots removed) — ✅ Phase 23
- [x] **PHOTO-03**: User can close the lightbox via the close button (fix carry-forward bug) — ✅ Phase 23

### Data Filters
- [x] **FILT-01**: User can filter occurrence records by county using a dropdown populated from the species' data — ✅ Phase 24
- [x] **FILT-02**: User can filter occurrence records by collection using a dropdown populated from the species' data — ✅ Phase 24
- [x] **FILT-03**: User can filter occurrence records by elevation range using a min/max slider (feet) — ✅ Phase 24
- [x] **FILT-04**: County, collection, and elevation filters update the map and phenology chart in real time — ✅ Phase 24

### Similar Species
- [x] **SIM-01**: Similar species section displays a thumbnail image for each similar species, loaded from CDN — ✅ Phase 25
- [x] **SIM-02**: Similar species thumbnail entries are clickable links to the respective species pages — ✅ Phase 25

## Future Requirements (carried forward to next milestone)

- Multi-select dropdowns for state and record type
- Subspecies filter (data exists in records but not exposed in UI)
- Month/day filters (low value for current dataset density)

---

## Out of Scope

| Feature | Reason |
|---------|--------|
| County/collection multi-select | Single-select sufficient for v2.1; multi-select adds implementation complexity |
| Subspecies, linked_photo filters | Low value for current data; deferred |
| Month/day discrete filters | Low value; year range slider covers the primary use case |
| Photographic plate link on species pages | Plates are not linked to species in current data model; deferred |
| Zoomify / deep-zoom viewer | Out of scope (established v1.0 decision) |
