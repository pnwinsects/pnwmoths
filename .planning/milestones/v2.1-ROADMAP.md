## Milestone Summary

**Key Decisions:**

- Chart.js v4 axis titles via `scales.{x,y}.title.{display,text}` — no Title plugin import needed; `beginAtZero: true` preferred over `min: 0`
- Sibling-walk inert pattern for lightbox focus trapping: walk from host to `<body>`, inert all siblings at each level except the ancestor chain — avoids `main.inert` self-blocking shadow DOM
- ResizeObserver guard: only update `_stripOverflows` when value actually changes (prevents infinite re-render loop in Lit)
- z-index 9000 on lightbox overlay (Leaflet zoom/attribution use z-index 1000; must exceed this)
- `min-width: 0` on CSS grid `1fr` children — otherwise cells expand past allocation to fit content
- Null elevation passthrough: `null < N` evaluates false in JS; no explicit null guard needed for default `elevationMin=0, elevationMax=15000` bounds
- Phenology chart stays in DOM with zero-height bars (not removed) when filter returns no results — avoids stale Chart.js instance on detached canvas
- Elevation slider uses `String()` coercion on Lit `.value` binding — prevents Lit from treating Number as a Lit property
- Similar species label: scientific name only (`<em>genus species</em>`), user-directed for taxonomic consistency
- Similar species section inside `.species-photos` div under carousel, user-directed for visual proximity

**Issues Resolved:**

- Lightbox close button self-blocked by `main.inert` — root cause missed in planning, caught at human checkpoint
- Lightbox obscured by Leaflet controls — z-index conflict not anticipated, caught at human checkpoint
- CSS grid cell expansion past 1fr — min-width: 0 fix required, caught at human checkpoint
- Phase 25 plan originally placed similar-species section after `.species-prose`; moved inside `.species-photos` during UAT

**Issues Deferred:**

- MAINT-03: build time under 5 min unverified on live CI
- WebP not enabled on bunny.net Optimizer
- No automated visual regression tests

**Technical Debt Incurred:**

- `.species-photos` div now contains both photo carousel and similar-species row; future phases modifying species.njk layout must be aware of this structure
