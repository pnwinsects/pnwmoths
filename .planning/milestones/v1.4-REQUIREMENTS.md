## Future Requirements (Carried Forward)

- srcset breakpoints beyond 2x — defer until real traffic data shows need
- Signed CDN URLs for access control — public site, no auth needed yet
- AVIF delivery — not supported by bunny.net as of April 2026

## Out of Scope

| Feature | Reason |
|---------|--------|
| S3-compatible upload CLI for bunny.net | bunny.net S3 in closed preview as of April 2026; rclone FTP is stable path |
| Custom CDN hostname (e.g. images.pnwmoths.org) | No domain name yet; *.b-cdn.net is fine for now |
| Local image proxy / imgproxy for dev | Not needed; all environments use CDN directly |
| GitHub LFS storage quota reclaim | Requires repo deletion + recreation; accept billing |
