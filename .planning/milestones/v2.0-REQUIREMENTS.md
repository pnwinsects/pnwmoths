## v2.0 Requirements (Final Status)

Requirements for the Glossary Tooltips milestone. All 11 requirements satisfied.

### Glossary Transform

- [x] **GLOS-01**: Build emits species prose HTML with the first occurrence of each glossary term wrapped in `<abbr class="glossary-term">` carrying `title`, `data-definition`, and `data-image-url` attributes — *Phase 19, verified 2026-04-23*
- [x] **GLOS-02**: Term matching is case-insensitive and whole-word only (no partial matches inside longer words) — *Phase 19, verified 2026-04-23*
- [x] **GLOS-03**: Terms containing regex metacharacters (e.g., `1A+2A`, `W-mark`, `CuA1`) are safely escaped before matching — *Phase 19, verified 2026-04-23*
- [x] **GLOS-04**: Only the first occurrence of each term per page is wrapped; subsequent occurrences are left as plain text — *Phase 19 (gap closure in 19-04), verified 2026-04-23*
- [x] **GLOS-05**: Transform runs only on species pages; the `/glossary/` page and browse pages are excluded — *Phase 19, verified 2026-04-23*
- [x] **GLOS-06**: `<abbr title="[definition excerpt]">` provides a no-JS degradation path so the definition is accessible without JavaScript — *Phase 19, verified 2026-04-23*

### Tooltip / Popover UI

- [x] **TIP-01**: Hovering or focusing a wrapped glossary term opens a popover panel showing the full definition text — *Phase 20, verified 2026-04-23*
- [x] **TIP-02**: Popover panel includes the CDN glossary image when the term has one (image URL from `data-image-url`); image-less terms show definition only — *Phase 20 (folded from Phase 21), verified 2026-04-23*
- [x] **TIP-03**: Popover is implemented via the native HTML Popover API with a small JS event-listener layer (~20 lines, no external library); it dismisses on mouseout/blur and via Escape — *Phase 20, verified 2026-04-23*

### Quality

- [x] **QA-01**: Automated unit tests cover: regex metacharacter escaping, first-occurrence deduplication per page, and prose-scope guard (glossary and browse pages are not transformed) — *Phase 19 (97/97 tests), verified 2026-04-23*
- [x] **QA-02**: Pagefind index verified to not include definition text (definition lives in `data-*` attributes, which Pagefind does not index) — *Phase 20, verified 2026-04-23*
