**src/components/parquet-cache.js:**
- Module-level `Map` cache keyed by slug.
- Export `loadParquet(slug)` async function that:
  - Returns cached result if present.
  - Otherwise fetches `/species/${slug}/records.parquet` via `asyncBufferFromUrl({ url })` from hyparquet.
  - Parses with `parquetReadObjects({ file })`.
  - Stores and returns the array of record objects.
- Export `filterRecords(records, filters)` pure function that:
  - Accepts `records` (array) and `filters` object `{ state, recordType, yearMin, yearMax }`.
  - Filters: if `state` is truthy and not "all", keep records where `r.state === filters.state`.
  - If `recordType` is truthy and not "all", keep records where `r.record_type === filters.recordType`.
  - If `yearMin` or `yearMax` set, keep records where `r.year >= yearMin && r.year <= yearMax`.
  - Returns the filtered array.
- Export `aggregateByMonth(records)` pure function:
  - Returns a 12-element array of counts (index 0 = Jan, index 11 = Dec).
  - For each record with `r.month >= 1 && r.month <= 12`, increments `counts[r.month - 1]`.

**src/components/pnwm-occurrence-map.js:**
- Import LitElement, html from 'lit'; import L from 'leaflet'; import 'leaflet/dist/leaflet.css'; import { loadParquet, filterRecords } from './parquet-cache.js'.
- Class `PnwmOccurrenceMap extends LitElement`.
- `createRenderRoot() { return this; }` — LIGHT DOM (Leaflet requirement).
- Static properties: `slug` (String), `filters` (attribute: false), `_records` (attribute: false, state: true), `_loading` (Boolean, state: true), `_error` (attribute: false, state: true).
- Constructor initializes: slug='', filters=null, _records=[], _loading=true, _error=null.
- `connectedCallback()`: call super, if slug is set, try loadParquet(slug), set _records, _loading=false. Catch errors: set _error, _loading=false.
- `render()`:
  - If _loading: return `html\`<div style="min-height:320px;display:flex;align-items:center;justify-content:center"><p style="color:var(--pico-muted-color)">Loading occurrence data...</p></div>\``
  - If _error: return `html\`<div style="min-height:320px;display:flex;align-items:center;justify-content:center"><p style="color:var(--pico-del-color)">Could not load occurrence data. Try reloading the page.</p></div>\``
  - Otherwise: return `html\`<div id="map-${this.slug}" style="min-height:320px" role="application" aria-label="Occurrence map for ${this.slug}"></div>\``
- `updated(changed)`: If changed has '_records' or 'filters', and not _loading, call `_renderMap()`.
- `_renderMap()`:
  - Apply filters: `const visible = this.filters ? filterRecords(this._records, this.filters) : this._records`.
  - Get container via `this.querySelector('[id^="map-"]')`.
  - If `this._map` exists, remove all layers then add tile layer back. Otherwise create map: `L.map(container).setView([46.5, -120.5], 5)` with OSM tile layer and attribution `'&copy; <a href="https://openstreetmap.org">OpenStreetMap</a> contributors'`.
  - Clear existing marker layer group if present.
  - Create circleMarkers for records with valid lat/lon: `L.circleMarker([r.latitude, r.longitude], { radius: 6, color: '#0172ad', fillOpacity: 0.7 })` with popup showing locality, state, county, year, month, collector, record_type.
  - Add to L.featureGroup, add to map, fitBounds with pad(0.1) if markers exist.
  - Store marker group ref for later clearing.
  - Call `this._map.invalidateSize()`.
  - If no markers: show empty state text in container.
- `disconnectedCallback()`: Remove map if exists (`this._map.remove()`), call super.

**src/components/pnwm-phenology-chart.js:**

**src/components/pnwm-filter-bar.js:**

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Parquet file -> browser | Untrusted data loaded from static file into DOM |
| OSM tile server -> browser | Third-party tile requests reveal user IP/location |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | pnwm-occurrence-map.js, pnwm-phenology-chart.js | mitigate | All Parquet string values (locality, collector, notes) rendered via Lit html tagged template literals which auto-escape. NEVER use innerHTML with Parquet data. |
| T-03-02 | Tampering | vite.config.js | accept | Parquet files served over HTTPS in production (hosting provider TLS). No client-side integrity hash mechanism available in hyparquet. Risk: low (files are build-time generated, not user-uploaded). |
| T-03-03 | Information Disclosure | pnwm-occurrence-map.js | accept | OSM tile requests expose user IP to tile.openstreetmap.org. Standard browser behavior for any map; no PII in tile URLs. Acceptable for a public research site. |
| T-03-04 | Spoofing | Leaflet tile layer | accept | Tile URL is hardcoded to tile.openstreetmap.org over HTTPS. DNS spoofing is out of scope for ASVS L1 static site. |
</threat_model>
