  <what-built>

    strips with 93px height, show/hide images checkbox, and state filter dropdown that mutes taxa
    with no records in the selected state.
  </what-built>
  <how-to-verify>
    1. Run `npm run build` if not already done, then open `_site/browse/index.html` in a browser
       (or serve with `npx serve _site` and visit http://localhost:3000/browse/).

    2. **BROWSE-02 — Families collapsed by default with images:**
       - Confirm all family names are visible (h2 headings)
       - Confirm families are collapsed (no subfamily/genus lists visible)
       - Confirm up to 4 thumbnail images appear next to each family name (when images toggle is on)
       - Images should be 93px tall, horizontally scrollable if many

    3. **BROWSE-03 — Expand family → reveals subfamilies or genera:**
       - Click a family button to expand it
       - Confirm family-level images disappear (hidden when expanded)
       - Confirm subfamilies appear (or genera directly if no subfamily for that family)
       - Confirm multiple families can be open simultaneously

    4. **BROWSE-04 — Expand subfamily → reveals genera with images:**
       - Click a subfamily button to expand it
       - Confirm subfamily images disappear when expanded
       - Confirm genus rows appear with images

    5. **BROWSE-05 — Expand genus → reveals species links:**
       - Click a genus button to expand it
       - Confirm species appear as `<em>Genus species</em> — Common Name` links
       - Confirm each link points to `/species/{slug}/` (hover to check URL)

    6. **BROWSE-06 — Navigation images present and correct size:**
       - DevTools: inspect any image strip element; confirm height is 93px
       - Confirm no broken image icons (all navImages should resolve)

    7. **Show/hide images toggle:**
       - Uncheck "Show images" — confirm all image strips disappear globally
       - Re-check — confirm all image strips reappear

    8. **SFILT-02 — State filter:**
       - Select a state from the dropdown (e.g., "WA")
       - Confirm some taxa become visually dimmed (opacity ~0.35) but remain in the DOM
       - Confirm taxa with WA records remain at full opacity
       - Select "All states" — confirm all taxa return to full opacity
  </how-to-verify>
