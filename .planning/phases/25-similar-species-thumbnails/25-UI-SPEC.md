## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Gap between thumbnails in the similar-species flex row (matches photo carousel `.thumbnail-strip` gap: 4px) |
| sm | 8px | Margin-top between thumbnail and name label |
| md | 16px | Default element spacing |
| lg | 24px | Section padding (`.content-wrapper` established value) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Thumbnail height: 93px (hard design requirement matching photo carousel strip and pnwinsects-app reference; not a multiple of 4 but a locked dimension from D-04 in CONTEXT.md)
- Gray placeholder box: 93px height × 60px width (60px chosen as a natural portrait-proportion stand-in; visually neutral)

Source: `src/styles/theme.css` body font, `pnwm-image-slideshow.js` `.caption-line`.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f3e8ba` | Page background, cream surface — established site token |
| Secondary (30%) | `#ffffff` | `.content-wrapper` box, image cards, popover surfaces |
| Accent (10%) | `#a4ab78` | Links on hover, active thumbnail border — `var(--pico-primary)` |
| Destructive | not applicable | No destructive actions in this phase |

Accent reserved for: link hover color (`nav a:hover`), active-thumbnail border in photo carousel (`.thumbnail[aria-selected="true"]`). **Not** applied to the similar-species section itself — links use Pico's default link color.

**Placeholder box color:** `#d6d0bc` — a desaturated warm neutral derived from the cream background (#f3e8ba) darkened ~15% to be visually distinct but not jarring. This is Claude's discretion (per CONTEXT.md). Satisfies D-06 (purely visual, no text).

**Thumbnail border on hover:** `1px solid var(--pico-primary)` (`#a4ab78`) — matching the photo carousel selected-thumbnail pattern. Applied via `:hover` on the `<a>` wrapping each entry.

## Component Inventory

This phase has no new JS components. It is a pure Nunjucks template change.

### New HTML Structure (replacing lines 83–96 of `species.njk`)

```
<section class="similar-species">
  <h2>Similar species</h2>
  <div class="similar-species-row">
    <!-- for each similar slug: -->
    <a class="similar-species-entry" href="/species/{slug}/">
      <div class="similar-species-thumb">
        <!-- if images[slug][0] exists: -->
        <img src="{cdnBaseUrl}/{slug}/{filename}" alt="{name}" height="93" loading="lazy">
        <!-- else: -->
        <div class="similar-species-placeholder" aria-hidden="true"></div>
      </div>
      <span class="similar-species-name">{common_name or genus + species}</span>
    </a>
  </div>
</section>
```

### New CSS Classes

| Class | Rule |
|-------|------|
| `.similar-species` | No additional style needed; inherits `<section>` spacing |
| `.similar-species-row` | `display: flex; gap: 4px; overflow-x: auto; scroll-behavior: smooth; scrollbar-width: none;` |
| `.similar-species-row::-webkit-scrollbar` | `display: none;` |
| `.similar-species-entry` | `display: flex; flex-direction: column; align-items: center; text-decoration: none; flex-shrink: 0;` |
| `.similar-species-entry:hover .similar-species-thumb img` | `outline: 2px solid var(--pico-primary);` |
| `.similar-species-thumb` | `height: 93px; display: flex; align-items: center;` |
| `.similar-species-thumb img` | `height: 93px; width: auto; display: block;` |
| `.similar-species-placeholder` | `height: 93px; width: 60px; background: #d6d0bc; border-radius: 2px;` |
| `.similar-species-name` | `font-size: 14px; color: var(--pico-color); text-align: center; margin-top: 8px; max-width: 80px; overflow-wrap: break-word;` |

**`object-fit` decision (Claude's discretion):** Use `object-fit: contain` — avoids cropping wing patterns and diagnostic markings that are visually critical for species identification. The photo slideshow also uses `object-fit: contain` in the lightbox. Source: CONTEXT.md discretion note + `pnwm-image-slideshow.js` lightbox img rule.

**`<ul>` vs `<div>` decision (Claude's discretion):** Use `<div class="similar-species-row">` containing `<a>` elements. A `<ul>` would require extra CSS to remove list-item markers and indentation, and the entries are navigational links (not list items). Using `<div>` is cleaner for a flex link-row pattern.

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Click on similar-species entry | Navigates to species page (standard `<a href>`) |
| Hover on similar-species entry | Thumbnail gets `outline: 2px solid var(--pico-primary)` — visual affordance that it is clickable |
| No JS | Section renders identically — it is entirely static HTML |
| Missing image | Gray placeholder box shown; name link still rendered below (D-07) |
| Many similar species | Row scrolls horizontally, no wrapping (D-03) |

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Not applicable — no button actions; entries are navigational links |
| Section heading | "Similar species" (matches current `<h2>` at line 84 of `species.njk`) |
| Empty state heading | Not applicable — the section only renders when `sp.similar_slugs` has entries (existing Nunjucks guard at line 83) |
| Empty state body | Not applicable |
| Error state | Not applicable — missing image falls back silently to placeholder; no user-visible error copy |
| Destructive confirmation | Not applicable — no destructive actions |

**Alt text for thumbnails:** `{s.genus} {s.species}` — the scientific name. Same pattern as the main slideshow `alt` attribute on line 43 of `species.njk`. Common name is not used for alt text (not always present).

**Placeholder aria:** `aria-hidden="true"` on the placeholder `<div>` — it is decorative and carries no information. The `<a>` link's text content (`similar-species-name`) describes the destination.

## Accessibility Contract

| Element | Requirement |
|---------|-------------|
| `<a class="similar-species-entry">` | Contains visible text (name label) — no `aria-label` needed |
| `<img>` thumbnail | `alt="{genus} {species}"` — descriptive, non-empty |
| `.similar-species-placeholder` | `aria-hidden="true"` — decorative only |
| Scrollable row | No `role="list"` needed; it is a group of links, not a list |
| Keyboard navigation | Standard tab order through `<a>` elements — no JS required |
| `loading="lazy"` | Applied to all similar-species thumbnails (below-the-fold content) |
