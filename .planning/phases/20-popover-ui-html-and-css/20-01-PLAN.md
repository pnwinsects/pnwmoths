Rewrite `src/components/glossary-tooltip.js` to use the native Popover API. The new file
replaces the entire contents. Here is the complete specification:

**Module docblock** -- preserve the existing docblock (lines 1-9) exactly as-is.

**Per-term popover creation** -- replace the single shared tooltip div (old lines 11-16)
with a loop that creates one `<div popover>` per `<abbr class="glossary-term">`:

```javascript
const terms = document.querySelectorAll('abbr.glossary-term');
let hideTimer;

terms.forEach((abbr, index) => {
  // Create per-term popover element
  const popover = document.createElement('div');
  popover.id = `gt-popover-${index}`;
  popover.className = 'glossary-popover';
  popover.setAttribute('popover', 'auto');
  popover.setAttribute('role', 'tooltip');
  popover.setAttribute('aria-hidden', 'true');
  popover.innerHTML = '<img class="gt-img" alt=""><p class="gt-def"></p>';
  document.body.appendChild(popover);

  const gtImg = popover.querySelector('.gt-img');
  const gtDef = popover.querySelector('.gt-def');
```

**title-to-aria-label swap + tabindex** -- inside the same forEach loop, preserve the
existing pattern and add tabindex (per D-06, D-09):

```javascript
  // Move title -> aria-label (D-09)
  const title = abbr.getAttribute('title');
  if (title) {
    abbr.setAttribute('aria-label', title);
    abbr.removeAttribute('title');
  }
  // Make keyboard-focusable (D-06)
  abbr.setAttribute('tabindex', '0');
```

**Event wiring** -- inside the forEach loop. Remove mousemove (no cursor-following).
Both mouseenter and focus use getBoundingClientRect positioning. Preserve 80ms debounce:

```javascript
  abbr.addEventListener('mouseenter', () => {
    clearTimeout(hideTimer);
    show();
  });
  abbr.addEventListener('mouseleave', () => {
    hideTimer = setTimeout(() => hide(), 80);
  });
  abbr.addEventListener('focus', () => {
    clearTimeout(hideTimer);
    show();
  });
  abbr.addEventListener('blur', () => {
    hideTimer = setTimeout(() => hide(), 80);
  });
```

**show function** -- defined inside the forEach closure so it captures `abbr`, `popover`,
`gtImg`, `gtDef`:

```javascript
  function show() {
    const imageUrl = abbr.dataset.imageUrl;
    const definition = abbr.dataset.definition;

    gtDef.textContent = definition || '';

    if (imageUrl) {
      gtImg.src = imageUrl;
      gtImg.hidden = false;
    } else {
      gtImg.src = '';
      gtImg.hidden = true;
    }

    // Position below the term (D-04, D-05)
    const rect = abbr.getBoundingClientRect();
    let left = rect.left + window.scrollX;
    const top = rect.bottom + window.scrollY + 6;

    popover.showPopover();
    popover.removeAttribute('aria-hidden');

    // Viewport edge clamp
    const popoverWidth = popover.offsetWidth;
    const viewportWidth = window.innerWidth;
    if (left + popoverWidth > viewportWidth - 8) {
      left = viewportWidth - popoverWidth - 8;
    }

    popover.style.left = left + 'px';
    popover.style.top = top + 'px';
  }
```

Note: `popover.showPopover()` must be called BEFORE reading `popover.offsetWidth` because
the element is hidden (display:none equivalent) until shown, so offsetWidth would be 0.

**hide function** -- defined inside the forEach closure:

```javascript
  function hide() {
    try { popover.hidePopover(); } catch (_) { /* already hidden */ }
    popover.setAttribute('aria-hidden', 'true');
  }
```

The try/catch around `hidePopover()` prevents errors when the Popover API has already
hidden the popover (e.g., user clicked outside, which auto-dismisses it). The `toggle`
event from the Popover API can also cause a race with the 80ms debounce timer.

**Close the forEach:**

```javascript
});
```
