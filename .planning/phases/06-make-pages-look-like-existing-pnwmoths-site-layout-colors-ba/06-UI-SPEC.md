# Phase 6 — UI Design Contract

> Visual and interaction contract for Phase 6: Make pages look like the existing pnwmoths.biol.wwu.edu site — layout, colors, banner image.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Reference Site

**Existing site:** https://pnwmoths.biol.wwu.edu/

Colors, typography, and layout were extracted directly from the existing site's
stylesheets (`/media/css/custom.css`, `/media/css_pages/home.css`).
The banner image is sourced from `/media/images/header.png` on the existing site —
a horizontal strip of moth specimens on a black background.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Pico CSS 2.1.1 via CSS custom property overrides) |
| Preset | not applicable |
| Component library | none — Pico CSS classless semantic HTML |
| Icon library | none |
| Font | Open Sans (Google Fonts) for body; Spinnaker (Google Fonts) for headings |

**Implementation approach:** Override Pico CSS custom properties in a `src/styles/theme.css`
file. Do NOT replace Pico. Extend it.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing (nav links) |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Banner image: `margin: 0` — flush to header edges, no inset
- Max content width: `1140px` — matches existing site grid width
- Header/footer background extends full viewport width; content constrained to `1140px` centered

---

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 16px | 400 | 1.7 | Open Sans, Verdana, sans-serif |
| Label / nav | 14px | 400 | 1.5 | Open Sans, Verdana, sans-serif |
| Heading (h2) | 20px | 600 | 1.2 | Spinnaker, Open Sans, sans-serif |
| Display (h1) | 28px | 600 | 1.2 | Spinnaker, Open Sans, sans-serif |

**Source:** Existing site uses `'Open Sans', Verdana, sans-serif` for body and
`'Spinnaker', 'Open Sans', sans-serif` for headers. Sizes are normalized to px from
the existing site's em/relative values.

**Google Fonts load:** Add a single `<link>` to `base.njk` loading both Open Sans (400, 600)
and Spinnaker (400) with `display=swap`.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f3e8ba` | Page background — warm cream/tan |
| Secondary (30%) | `#ffffff` | Content wrapper, cards, body text areas |
| Accent (10%) | `#a4ab78` | Navigation highlights, dividers, selected state borders |
| Header / Footer surface | `#000000` | Header and footer backgrounds |
| Header / Footer text | `#ffffff` | Text and links inside header/footer |
| Destructive | `#D15E25` | Error states only (orange, matches existing FAQ header accent) |

Accent reserved for:
- Active/selected navigation item background or underline
- Horizontal rule / section dividers
- Border on hover states for species cards

**Color sources:** Extracted from `pnwmoths.biol.wwu.edu/media/css/custom.css`:
`#f3e8ba` (page bg), `#a4ab78` / `#a2aa77` (accent greens), `#000000` (header/footer),
`#D15E25` (orange accent).

**Pico CSS overrides required:**
```css
:root {
  --pico-background-color: #f3e8ba;
  --pico-primary: #a4ab78;
  --pico-primary-hover: #7f8956;
  --pico-font-family: 'Open Sans', Verdana, sans-serif;
  --pico-font-family-sans-serif: 'Open Sans', Verdana, sans-serif;
}
nav {
  background-color: #000000;
}
```

## Layout Contract

### Header

- Full-width black (`#000000`) background
- Banner image centered horizontally, `max-width: 1140px`, zero vertical margin
- Banner image source: download from `https://pnwmoths.biol.wwu.edu/media/images/header.png`
  and commit to `src/images/header.png`
- Banner `alt="Pacific Northwest Moths — a horizontal strip of moth specimens"`
- Below the banner: horizontal navigation bar (still black background), links in white

### Navigation bar

- Background: `#000000`
- Links: `#ffffff`, no underline at rest
- Active page link: `#a4ab78` background pill or underline
- Padding: `8px 16px` per nav link
- Site title text not repeated if banner image is present (banner serves as the site identity)

### Main content

- Background: `#f3e8ba` (cream) for the outer page
- Content wrapper (centered, `max-width: 1140px`): `#ffffff` background with `box-shadow: 0 0 15px 5px rgba(50,50,50,0.25)`
- Padding inside wrapper: `24px`

### Footer

- Background: `#000000`
- Text: `#ffffff`
- Font size: 14px
- Content: licensing statement (Creative Commons Attribution-NonCommercial-ShareAlike 4.0) and institutional credits
