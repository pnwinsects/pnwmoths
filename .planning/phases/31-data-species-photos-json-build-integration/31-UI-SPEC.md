# Phase 31 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Phase Classification

**UI surface introduced:** Minimal. Phase 31 is a data pipeline phase. The sole template
change is a one-line Nunjucks guard that suppresses low-res `<figure>` elements on species
pages where `high_res_available: true`. No new visual components, no new layouts, no new
interactive elements are introduced.

**What changes visually:**
- Species pages with high-res photos: the existing `<pnwm-image-slideshow>` web component
  receives `high-res-available` and `high-res-specimens` attributes, but the `<figure>` slot
  is intentionally empty (Phase 32 fills it with OSD viewer). Until Phase 32 ships, these
  species have no photo display in their lightbox. This is a known intermediate state — the
  template guard is correct; Phase 32 completes the experience.
- Species pages without high-res: unchanged — low-res `<figure>` elements render exactly
  as they did after Phase 23 (carousel + lightbox).

**What does NOT change:** Color palette, typography, spacing, navigation, layout,
interactive behavior, error states, empty states, destructive actions, or any component
not named above.

## Template Guard (DATA-03)

The only template change in Phase 31 is a single Nunjucks conditional in
`src/species/species.njk` line 47.

**Before:**
```nunjucks
{% if spImages and spImages.length > 0 %}
```

**After (D-04 exact wording from CONTEXT.md):**
```nunjucks
{% if (not (highResEntry and highResEntry.high_res_available)) and spImages and spImages.length > 0 %}
```

This suppresses the `<figure>` loop entirely for species where `high_res_available` is
`true`. No CSS changes are needed. No new HTML elements are introduced.

**Visual result for high-res species:** The `<pnwm-image-slideshow>` component is still
rendered in the DOM (it receives `high-res-available` and `high-res-specimens` attributes),
but the `<figure>` slot is empty. Phase 32 adds OSD thumbnail rendering into this slot.

**Visual result for non-high-res species:** Unchanged — existing carousel and lightbox
behavior from Phase 23 are unaffected.
