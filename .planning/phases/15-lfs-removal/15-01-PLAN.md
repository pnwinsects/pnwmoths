**Step 1 — Install git-filter-repo (not currently installed):**

**Step 2 — Record current state for reference:**

**Step 3 — Create fresh clone using --no-local (CRITICAL: must use --no-local):**
```bash
cd /tmp
git clone --no-local https://github.com/pnwinsects/pnwmoths.git pnwmoths-lfs-rewrite
cd /tmp/pnwmoths-lfs-rewrite
```

The `--no-local` flag prevents Git from using filesystem hardlinks, which would corrupt
the original repo's packfiles if filter-repo prunes objects.

**Step 4 — Run filter-repo to excise both paths from all history:**
```bash
git filter-repo --invert-paths --path images/ --path plates/
```

Multiple `--path` flags with `--invert-paths` form a union — any commit touching images/
or plates/ is rewritten to exclude those paths. filter-repo automatically prunes reflogs
and runs gc after the rewrite. This operates on 130-byte LFS pointer files only (no LFS
object download required per D-03).

**Step 5 — Delete .gitattributes (all 4 lines are LFS-only; file has no other content):**
```bash
git rm .gitattributes
```

**Step 6 — Update .gitignore to add images/ and plates/ (per D-05):**

images/
plates/
