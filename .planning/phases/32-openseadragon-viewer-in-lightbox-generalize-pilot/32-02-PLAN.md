Add in-lightbox prev/next specimen navigation to `pnwm-image-slideshow.js` (D-07 / D-08) for species with multiple high-res specimens; investigate and fix the close-button issue folded from Phase 23; add unit tests for the new navigation logic, the `useOsd` derivation (low-res regression check), and the view-to-label mapping (Wave 0 gaps from RESEARCH.md).
